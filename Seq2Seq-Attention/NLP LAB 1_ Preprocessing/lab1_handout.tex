\section{Objective}
Understand preprocessing steps required for Neural Machine Translation:
\begin{itemize}
\item Tokenization
\item Vocabulary construction
\item Sentence encoding
\item Subword tokenization concept (BPE)
\end{itemize}

\section{Dataset}

\begin{verbatim}

\section{Task 1: Tokenization}

Example:
\begin{verbatim}
Input:
"I am a student"

Output:
["i","am","a","student"]
\end{verbatim}

\begin{lstlisting}[language=Python]
def tokenize(sentence):
    return sentence.lower().split()
\end{lstlisting}

\section{Task 2: Vocabulary}

Example mapping:

\begin{verbatim}
<PAD> -> 0
<BOS> -> 1
<EOS> -> 2
i -> 3
am -> 4
student -> 5
\end{verbatim}

\section{Task 3: Sentence Encoding}

Example:

\begin{verbatim}
Sentence:

Encoded:

\end{verbatim}
